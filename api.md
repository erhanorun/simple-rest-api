# Create User

Post user with `username`.

**URL** : `/api/user`

**Method** : `POST`

**Auth required** : `No`

**Permissions required** : `None`

**Headers** :
```json
{
    "Content-Type": "application/json"
}
```

## Success Response

**Code** : `200 OK`
